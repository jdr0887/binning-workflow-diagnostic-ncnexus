RESTful Services

Submit (POST)

http://<host>:<port>/cxf/DiagnosticNCNEXUS/DiagnosticNCNEXUSService/submit

The 'submit' service accepts a JSON string containing the following keys:

  • participant
  • gender
  • dxId
  • listVersion

For example:

{"diagnosticBinningJobInfo":{"participant":"test","gender":"M","dxId":46,"listVersion":40}}

The response will include the binning job identifier as a long.

Lookup Status (GET)

http://<host>:<port>/cxf/DiagnosticNCNEXUS/DiagnosticNCNEXUSService/status/{binningJobId}

The response will be a deserialization of a DiagnosticStatusType object. The reponse format can be rendered as xml or json (defaults to xml). For example:

http://<host>:<port>/cxf/DiagnosticNCNEXUS/DiagnosticNCNEXUSService/status/1234.xml
http://<host>:<port>/cxf/DiagnosticNCNEXUS/DiagnosticNCNEXUSService/status/1234.json

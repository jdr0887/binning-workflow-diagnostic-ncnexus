Create Job

The create binning job will take the arguments and persist a DiagnosticBinningJob with a 'NCNEXUS' study & return the identifier.

karaf@root()> diagnostic-ncnexus:create-binning-job --gender M --participant NCG_00020 --listVersion 22 --dxId 7
4246

Load VCF

Persist variants from a vcf file...skipping homozygous reference calls.

karaf@root()> diagnostic-ncnexus:load-vcf --binningJobId 4246

Annotate Variants

Annotating a LocatedVariant involves determining the transcript location & adding HGVS expressions.

karaf@root()> diagnostic-ncnexus:annotate-variants --binningJobId 4246

Load Coverage

Coverage can be from a GATKDepthOfCoverage summary file or from a SAMToolsDepth file.

karaf@root()> diagnostic-ncnexus:load-coverage --binningJobId 4246

Update Frequencies

Updating the MaxFrequency table to based on information from 1000Genome, ExAC, or dbSNP.

karaf@root()> diagnostic-ncnexus:update-frequencies --binningJobId 4246

Update Diagnostic Bins

Do the actual binning using MaxFrequency, HGMD, and/or ClinVar metadata.

karaf@root()> diagnostic-ncnexus:update-diagnostic-bins --binningJobId 4246

Generate Report

Persists counts from tables involved in running the binning process (ie, # of snps processed, # of HGDM known pathogenic variants, etc.).

karaf@root()> diagnostic-ncnexus:generate-report --binningJobId 4246
